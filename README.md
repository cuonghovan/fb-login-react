# react-redux-demo

React redux application demos.
This project uses some additional helpful libraries:

* Immutable.js
* Redux-saga
* Axios

## Install

### Requirement

* Install Node.js and npm https://nodejs.org

### Setup for Development build

Step1:

```
npm install
```

Step 2:

```
npm start
```

### Setup for Production build

Step 1:

```
npm install
```

Step 2:

```
npm run build
```
